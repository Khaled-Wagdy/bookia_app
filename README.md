# 📚 Bookia Store App

تطبيق **Bookia** لبيع الكتب، مصمم بواجهات عصرية باستخدام **Flutter** بناءً على تصاميم **Figma** الاحترافية. يركز المشروع على الكود النظيف (Clean Code) واستخدام الـ **Custom Widgets** لضمان عدم التكرار وسهولة الصيانة.
